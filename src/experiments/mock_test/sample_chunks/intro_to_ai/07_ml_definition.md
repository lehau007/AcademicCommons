# Defining a Machine Learning Problem

According to Mitchell (1997), a machine learning problem is defined as improving with experience at some task. Specifically, it requires improving over a task T, with respect to a performance measure P, based on experience E. This TPE framing is broad enough to cover supervised, unsupervised, and reinforcement learning settings.

Information gain is a statistical measure that calculates the expected reduction in entropy caused by partitioning instances according to a specific attribute. In Decision Tree learning, information gain is used to determine which attribute is most useful for classifying training instances at each node, with the algorithm selecting the attribute that yields the highest information gain. This greedy criterion underlies ID3 and C4.5.
